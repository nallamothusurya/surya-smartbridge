## Getting Started

### Prerequisites

- Python 3.x
- Flask

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/nallamothusurya/surya-smartbridge.git
   cd surya-smartbridge
   ```

2. **Install dependencies:**
   ```bash
   pip install flask
   ```

3. **Run the app locally:**
   ```bash
   python app.py
   ```
   The app will be available at `http://127.0.0.1:5000/`.
